Tips en tricks

- Krijg je een melding dat het bestand van een onbekende ontwikkelaar is en dus niet geopend kan worden,
klik dan met de rechtermuisknop op het bestand en dan op 'open'. Op deze manier krijg je wel een optie om de app op te starten
en daarna zal de waarschuwing niet meer verschijnen.

- Als de Windows versie niet opstart kan dat zijn omdat er geen java op de gebruikte laptop staat.
Download daarvoor de nieuwste java versie op de oracle site (zie link bij download)
